# 🧪 QA Manual - SwagLabs Shopping

## 📖 Sobre o projeto

Este projeto tem como objetivo demonstrar, na prática, o dia a dia de um Analista de QA com foco em testes manuais funcionais, utilizando uma aplicação de e-commerce fictícia (SwagLabs Shopping).

O projeto foi desenvolvido com base em conceitos de metodologias ágeis, contemplando levantamento de requisitos, escrita de User Stories, planejamento de testes e execução de cenários.

---

## 🎯 Objetivo

Aplicar na prática:

- Levantamento e análise de requisitos
- Escrita de User Stories
- Definição de critérios de aceite
- Planejamento de testes
- Criação de casos de teste
- Simulação do fluxo de trabalho de QA
- Documentação de testes

Conforme apresentado no material do curso, o foco está em testes:
- Manuais
- Funcionais
- Nível de sistema e aceite :contentReference[oaicite:0]{index=0}
